## 네트워크 설계

### VPC CIDR 구성

| Subnet | CIDR | 용도 | AZ |
|--------|------|------|-----|
| Public-a | 10.0.1.0/24 | ALB, NAT Instance | ap-northeast-2a |
| Public-c | 10.0.2.0/24 | ALB (HA) | ap-northeast-2c |
| Private-a | 10.0.10.0/24 | EKS, ECS | ap-northeast-2a |
| Private-c | 10.0.11.0/24 | EKS, ECS | ap-northeast-2c |
| Database-a | 10.0.20.0/24 | RDS, ElastiCache | ap-northeast-2a |
| Database-c | 10.0.21.0/24 | RDS, ElastiCache | ap-northeast-2c |

### Security Groups

| SG | 인바운드 | 용도 |
|----|----------|------|
| alb-sg | 80, 443 (0.0.0.0/0) | 외부 트래픽 |
| eks-nodes-sg | All (alb-sg, ecs-sg) | EKS 노드 |
| ecs-tasks-sg | 8000 (eks-sg) | AI 서비스 |
| rds-sg | 5432 (eks-sg, ecs-sg) | PostgreSQL |
| redis-sg | 6379 (eks-sg, ecs-sg) | Redis |
| nat-sg | All (Private CIDR) | NAT Instance |

### S3 버킷 구성

| 버킷 | 용도 | Lifecycle | 비고 |
|------|------|-----------|------|
| goormgb-static | Next.js 정적 파일 | - | CloudFront OAC |
| goormgb-images | 일반 이미지 | - | CloudFront OAC |
| goormgb-logs | 로그 | 30d→IA, 90d→Glacier, 1y→삭제 | |
| goormgb-backup | 일반 백업 | 30d→Glacier, 1y→삭제 | |
| **goormgb-ai-trajectory** | 궤적 데이터 아카이브 | 7d→Glacier IR, 1y→Deep, 3y→삭제 | MongoDB 백업 |
| **goormgb-ai-vqa-data** | VQA 퀴즈/결과 아카이브 | 30d→IA, 180d→Glacier IR | MongoDB 백업 |
| **goormgb-ai-vqa-images** | VQA 퀴즈 이미지 | - | CloudFront 연동 가능 |
